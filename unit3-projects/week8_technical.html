Bubble Sort

The Idea: compare the items next to each other in the array. Switch them if needed, then continue down the array. It's called bubble sort becasue the elements in the array "bubble" to their spot.


Below are exact steps for the algorithm:

  1. Look the first element in the array. If it is larger than the next element in the array, swap the two numbers.

  2. Move to the next index in the array and do the same comparison as in step 1.

  3. Repeat step 2 until you reach the end of the array.

  4. When you reach the end of the array, go back to the beginning and start at step 1 again.

  5. Repeat until all items are sorted.

Merge Sort

An important concept in merge sort, and sorting in general, is that an array of length 1 is sorted. For example, the array [2] is sorted. This may seem basic, but it is the foundation of the next two sorting algorithms.

Before jumping into merge sort, let's think about merging two arrays that are already sorted.

Array1 = [1, 3, 5, 9]   Array2 = [2, 4, 6, 8]

To merge them and sort them at the same time:

  1. create an empty array called sorted_array

  2. compare the first element in each array

  3. add the smaller element to sorted_array

  4. repeat steps 2-3 until all items are in sorted_array

Array2 = [2, 4, 6, 8]

sorted_array = []

Array2 = [2, 4, 6, 8]

Array2 = [4, 6, 8]

sorted_array = [1, 2]

Array2 = [4, 6, 8]

sorted_array = [1, 2, 3]

Array2 = [6, 8]

sorted_array = [1, 2, 3, 4]


Array1 = [9]

Array2 = [6, 8]

sorted_array = [1, 2, 3, 4, 5]


Array1 = [9]

Array2 = [8]

sorted_array = [1, 2, 3, 4, 5, 6]


Array1 = [9]

Array2 = []

Sorting complete!



Now we know how to take two sorted arrays and merge them into one sorted array. So how does this help us with an unsorted array like [5,2,7,8,1,9,3,6]?

Remember what I said in the beginning? All arrays of length one are sorted! So turn the unsorted array into a combination of sorted arrays.

The unsorted array [5,2,7,8,1,9,3,6] now becomes a combination of 8 sorted arrays of length one!

[5] + [2] + [7] + [8] + [1] + [9] + [3] + [6]

Now go through two by two and merge the arrays, like we did above. The arrays become:

[2,5] + [7,8] + [1,9] + [3,6]

Look at that! Now we have four arrays that are sorted. Do you see where this is going? Now go through two by two and merge the arrays.

[2,5,7,8] + [1,3,6,9]

Now we have 2 sorted arrays. Merge them one last time.

[1, 2, 3, 5, 6, 7, 8, 9]

Now we have a sorted array!

Quick Sort

Oh, quick sort, you are so cool, but so hard to explain. When I learned quick sort in class, my professor took a whole class period and made us act it out multiple times so we would fully understand it.

The idea: Assign a number in the array the role of "pivot". Go through the array, put all elements smaller than it to the left of the pivot and put all numbers larger than it to the right of the pivot. Now the pivot is in the exact right spot with two unsorted arrays on either side.

Call this algorithm again on the arrays on either side of the pivot. Now both of these arrays will get a pivot and the formula continues.

Keep calling this formula on smaller and smaller arrays until each array is one item long, and thus sorted.


Step 1: Randomly assign pivot
Step 2: Put all # below pivot to the left of the pivot and all # above pivot to the right of the pivot
Step 3: Now the pivot is in the perfect spot. Split into smaller arrays
Step 4: Repeat
