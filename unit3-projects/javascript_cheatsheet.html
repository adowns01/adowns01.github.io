Conditionals

if, else, if else

if (hour < 12){
  greeting = "Good Morning";
}


if (hour < 12){
  greeting = "Good Morning";
} else {
  greeting = "Good Evening";
}


if (hour < 12) {
  greeting = "Good Morning";
} else if (hour < 18)
  greeting = "Good Afternoon";
} else {
  greeting = "Good Evening";
}

Switch

switch (new Date().getDay()) {
case 6:
  text = "Today is Saturday";
  break;
case 0:
  text = "Today is Sunday";
  break;
default:
  text = "Looking forward to the Weekend";
}

Loops

For

For loops should be used when you know exactly how many times you need to loop.

for (var i = 0; i < 5; i++) {
  console.log(i)
}
// ==> 1, 2, 3, 4

// if you already have an iterator defined, no need to make a new one
var i = 2;
for (; i < 5; i++) {
  console.log(i)
}
// ==> 2, 3, 4

While and Do/While

While loops should be used when you looping until a condition is satisfied.

while (condition == false){
  //do things
}

// use a do/while loop if the loop should always run at least once
do {
  // do things
}
while (condition == false)

Continue

When the continue statement is called, the loop immediately skips to the next iteration.

for (var i = 0; i < 4; i++) {
  if (i == 1) continue;
  console.log(i);
}
// ==> 0, 2, 3

Break

When the break statement is called, the loop immediately stops

for (var i = 0; i < 4; i++) {
  if (i == 1) {break;}
  console.log(i)
}
// ==> 0

Objects

Creating Objects

Literal Notation

var myObject = {
  property1: value1,
  property2: value2
};

Constructor

var myObject = new Object();
myObject["property1"] = value1;
myObject.property2 = value2;

Addings Functions to Objects

During object construction

// Note: I created an object in an object here
car.gasTank = {
  capacity: 20,
  contents: 10,
  fillUp: function(){
    this.contents = this.capacity;
  }
};

After object is created

// First create an object
var car = {
  color: "red",
  year: 2016,
  make: "Toyota",
  model: "Corola"
};

// Then add a function
car.showColor = function(){
  return this.color;
};

// Then call the function
car.showColor()

Iterating through an Object

for (property in myObject){
  console.log("The property " + property + " has value " + object[property]);
};
