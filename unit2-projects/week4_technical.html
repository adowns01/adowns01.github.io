Week 4 Technical Blog: What is Enumerable#group_by?

Let's start at the beginning.

Enumerable is a way to describe a class that can iterate through all of its elements. For example an array is enumerable, while strings and integers are not enumerable. So Enumerable#group_by means that this function is only valid for enumerable classes.

The hashtag, indicated that this is a public method. This means that you call if using dot notation. For instance:

my_array = [1,2,3]
my_array.group_by{#stuff goes here}

The purpose of Enumerable#group_by? is to sort objects by a certain parameter. For example you could sort numbers by if they are even or odd by seeing if they are divisible by 2:

my_array = [9,4,6,8,10,2,3]
my_array.group_by{|element| element % 2}
#=>{1=>[9, 3], 0=>[4, 6, 8, 10, 2]}

Or you could sort words in a sentence by length:

my_array = ["I", "cannot", "wait", "to", "attend", "DBC", "!!!"]
my_array.group_by{|element| element.length}
{1=>["I"], 6=>["cannot", "attend"], 4=>["wait"], 2=>["to"], 3=>["DBC", "!!!"]}
